Comment ça fonctionne ?

Le portefeuille virtuel permet de régler les frais de covoiturage directement sur la plateforme, en dollars canadiens.

1. Inscription au trajet : votre solde est débité immédiatement dès que vous rejoignez un trajet.

2. Confirmation conducteur : après le trajet, le conducteur confirme votre présence sur la plateforme.

3. Validation admin : l'administrateur valide définitivement le paiement → le conducteur est alors crédité.

Recharge du solde : l'administrateur recharge les soldes (règlement par virement, chèque ou espèces hors plateforme).
